Наши услуги

Предпроектный анализ

Обсуждаем идею, формируем концепцию продукта: описание, цели, задачи, аудитория. Анализируем бизнес-модель, риски, перспективы и ресурсы. Создаём прототипы и планируем реализацию.

Web-разработка

Разрабатываем сложные (в том числе highload) сайты и приложения. Наш код гибкий, лаконичный и понятный, обеспечивающий долгосрочное развитие продукта.

DevOps

Аудитируем инфраструктуру и процессы разработки, делаем миграцию в Kubernetes, внедряем CI/CD, разворачиваем СУБД и другие программные средства, настраиваем логи, метрики и мониторинг для приложений.

Аналитика

Выявляем и собираем ключевые метрики продукта и инфраструктуры, проектируем системы их сбора, обработки и анализа, понятно визуализируем для заказчика.

Проектирование

Анализируем требования, описываем архитектуру, выбираем инструментарий, создаём интерфейс и бизнес-логику для разработки, тестирования и внедрения системы. Результат — техническое задание.

Мобильная разработка

Разрабатываем нативные и кроссплатформенные мобильные приложения. Создаём сайты с адаптивным дизайном, оптимизируем производительность и улучшаем пользовательский опыт.

Десктопная разработка

Специализируемся на создании прикладного софта и установщиков для Windows, занимаемся разработкой кроссплатформенных десктопных приложений на основе Electron.js.

UX/UI-дизайн

Пропускаем через себя потребности заказчика и создаём привлекательные, лёгкие в использовании, функциональные интерфейсы.

Tехнологический стек

Серверная инфраструктура и CI/CD

Kubernetes/Helm/Terraform, Linux Debian/Ubuntu, Amazon S3, Google Cloud Platform, Nginx, Ansible, GitLab, Docker.

Frontend

Язык Javascript/ES6 с фокусом на React-экосистему: React.js, Redux, Material UI, Next.js (SSR, ISR, SSG), Styled Components.

Backend

Язык Python с фреймворками Django, FastAPI. Язык Node.js с собственным фреймворком поверх популярных пакетов. Брокеры сообщений AMQP/RabbitMQ.

Базы данных

MySQL, PostgreSQL, Redis, MongoDB, InfluxDB, Tarantool, Clickhouse.

Мониторинг и визуализация

Grafana, Loki, Prometheus/VictoriaMetrics.

Инструменты UX/UI

Прототипирование в Figma, подготовка макетов в Photoshop, иллюстрации в Illustrator, анимация в After Effects и другое.

Технологии кроссплатформенной разработки

React Native, Flutter для мобильных приложений. Electron.js, Chromium Embedded Framework (CEF) для десктопных приложений. Progressive Web Apps (PWA).
